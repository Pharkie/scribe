📝 Memos

Available Placeholders

[date] → Current date (24Aug25)

[time] → Current time (12:30)

[weekday] → Day name (Sunday)

[uptime] → Device uptime (2h13m)

[ip] → Device IP address

[mdns] → mDNS hostname

[coin] → Heads or Tails

[dice:n] → Random 1-n (default 6)

[pick:a|b|c] → Random choice
